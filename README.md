## 这是什么

**zero2PM** 是一个面向零基础转行者和初学者的产品经理系统化入门教程，目标是帮助想做产品但不知道从哪里开始的人，真正建立起产品经理的核心能力。

内容不停留在方法论堆砌和名词解释，而是从实际工作场景出发，覆盖需求分析、产品设计、数据驱动决策、项目管理，最终落到一个完整的产品实战项目。

**在线阅读（GitHub Pages）**：[https://onefly.top/zero2PM](https://onefly.top/zero2PM)

## 模块概览

| 模块 | 状态 | 内容 |
|------|------|------|
| PM 基础 | 🔲 待开始 | 产品经理角色认知、产品思维、用户需求 vs 业务需求、行业差异 |
| 产品设计 | 🔲 待开始 | 需求采集与分析、用户画像、信息架构、PRD 撰写、原型设计 |
| 产品策略 | 🔲 待开始 | 商业模式画布、竞品分析、市场定位、MVP 与 PMF |
| 数据分析 | 🔲 待开始 | 指标体系、漏斗分析、AB 测试、SQL 基础、数据看板 |
| 项目管理 | 🔲 待开始 | 敏捷开发、排期管理、跨部门协作、上线复盘 |
| 用户增长 | 🔲 待开始 | AARRR 漏斗、留存策略、裂变拉新、运营活动设计 |
| PM 面试 | 🔲 待开始 | 简历撰写、作品集、高频面试题、Case Study |
| 工具箱 | 🔲 待开始 | Figma、SQL、飞书/Jira、数据分析工具 |
| 实战项目 | 🔲 待开始 | 完整产品从 0 到 1 |

## 适合谁

- 想转行做产品经理，但不知道从哪里开始
- 应届生想入行 PM，需要系统化学习路线
- 技术/运营/设计背景，想理解产品经理的思维方式
- 准备产品经理面试，需要体系化复习
- 想把产品 Sense 从"感觉"变成"方法"

## 本地运行

```bash
git clone https://github.com/ranxi2001/zero2PM
cd zero2PM

# 安装 Jekyll（需要 Ruby）
gem install bundler jekyll
bundle install

# 本地预览
bundle exec jekyll serve
# 访问 http://localhost:4000/zero2PM
```
